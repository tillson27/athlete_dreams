# UI Redesign Skill

Completely rethink and redesign the UI from the ground up.

## Core Principles

1. **Preserve functionality** — All features and behavior must remain intact unless the user explicitly says otherwise.
2. **Full creative freedom** — Rethink layout, organization, spacing, typography, colors, and visual hierarchy. Nothing is sacred except the functionality.
3. **Professional and elegant** — Aim for polished, modern UI that feels intentional and cohesive.

## User Feedback

Treat user feedback as **general direction and ideas**, not strict requirements. Use it to inform your vision, but don't let it limit your creative decisions. Your job is to deliver the best possible UI—trust your judgment.

## Workflow

1. **Understand** — Read the target component(s) to understand what they do.
2. **Redesign** — Reimagine the UI completely. Consider:
   - Visual hierarchy and information architecture
   - Spacing, alignment, and rhythm
   - Typography and color usage
   - Component organization and layout
   - Micro-interactions and polish
3. **Implement** — Make the changes.
4. **Verify** — Ensure functionality still works as expected.
